# My personal website made with Gatsby JS
## 🚀 Quick start
### Develop

Navigate into your new site’s directory and start it up.

```shell
cd iltumio.dev/
npm run develop
```

### Serve

Navigate into your new site’s directory and start it up.

```shell
cd iltumio.dev/
npm run serve
```
